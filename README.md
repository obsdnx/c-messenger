# Terminal Messenging App - c-chat

c-chat is written fully in c++, i tried writing in python but not being able to see 
the memory management and data types got frusturating and prevented proper understanding.
python is just a c-wrapper anyway might as well suffer through the memory leaks.

##core systems 
client sends message to server, server echoes to all other connected clients, repeat until connection closes

##features (in sha Allah)
- Logs chat messages from Discord and Slack 
- Search messages by user or keyword

##core concepts
socket programming, TCP,  multi-threading
